# F1-Analytics-Dashboard

WEBSITE LINK : https://v-33.github.io/F1-Analytics-Dashboard/
🏎️ F1 Analytics Dasboard

A multi-role Formula 1 analytics platform built with HTML, CSS, and JavaScript.
The site provides interactive views for Drivers, Team Managers, and Fantasy Players, giving users access to F1 data, team management features, and fantasy insights—all in one platform.

Features
Role-based Login – Access the site as a Driver, Team Manager, or Fantasy Player.
Driver Dashboard – View personal stats, contracts, and performance insights.
Team Manager Dashboard – Manage team drivers, view contracts, and track team performance.
Fantasy Player Dashboard – Access driver stats and race data to make fantasy picks.
LocalStorage-backed Contracts – Simulates a backend to store, update, and track driver contracts.
Interactive UI – Built with HTML, CSS, and JavaScript; responsive and intuitive.
Lightweight & Easy to Run – No server setup required; just open index.html in a browser.

Installation & Usage
Clone the repository
git clone https://github.com/V-33/F1-Analytics-Dashboard.git
cd F1-Analytics-Dashboard
Open the site
Open index.html in any modern web browser.
Choose your role (Driver, Team Manager, or Fantasy Player) and login.
Explore
Interact with dashboards and contracts.
All data is stored in browser localStorage, so changes persist per session.

Tech Stack
HTML – Structure of pages
CSS – Styling, layout, and responsiveness
JavaScript – Frontend logic and role-based functionality
localStorage – Simulated backend for contract management
